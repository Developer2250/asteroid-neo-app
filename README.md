# Asteroid-neo-app

## Installation

1. Clone the project

```bash
https://github.com/Developer2250/asteroid-neo-app.git 
```

2. Checkout the branch
``` 
git checkout master
```

3. Install composer using composer commad
```
composer install
```

4. copy the .env.example file and create new file called ".env" and run the project
```
php artisan serve
```
